ABOUT altalanos_template
------------

altalanos_template is the default theme for Drupal 8.  It is a flexible, recolorable theme
with a responsive and mobile-first layout, supporting 16 regions.

The altalanos_template theme is named after Jean altalanos_template, one of the original programmers
for the ENIAC computer.

This theme is not intended to be used as a base theme.

To read more about the altalanos_template theme please see:
https://www.drupal.org/docs/8/core/themes/altalanos_template-theme
